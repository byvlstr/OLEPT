# OLEPT
"Optimization for Lyon's Ecological Public Transport"

Personal mini project by vlstr

Good estimation of project length : 108%

Tools and Technology
---
+ Java & XML

+ Eclipse & Notepad++



Features
---
+ Compute the itinerary its duration from one stop to another 

+ Choice of an "axis" to reduce the perimeter

+ Display the itinerary's stop list and the duration in minutes
